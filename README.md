# 🔐 Privacy Policy - Authra

This repository hosts the official **Privacy Policy** page for the mobile application **Authra**, developed by **TriVisionTech**.

Authra is a privacy-first two-factor authentication (2FA) app designed to operate **100% offline**, ensuring maximum security and user control.

## 🔐 Key Privacy Highlights

Authra is built with a strong commitment to privacy and security:

* No user accounts required
* No personal data collection
* No analytics or tracking
* No internet connection required
* No data transmission to external servers

---

## 📱 How Authra Handles Data

* All authentication tokens are stored **locally on the user's device**
* Data **never leaves the device**
* No cloud sync or remote backup is performed
* Users maintain **full control over their data at all times**

---

## ⚠️ Important Notes

* Since Authra does not store data externally, **data recovery is not possible** if the device is lost or reset
* Users are responsible for securing their own devices (PIN, biometrics, etc.)
